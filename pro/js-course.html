JavaScript Tags and Methods

JavaScript – Interactive Web Language

What is JavaScript?

JavaScript (JS) is a powerful scripting language used to make web pages interactive and dynamic. It allows you to control HTML and CSS, create animations, handle events, and build full web apps in the browser.

Uses of JavaScript:

  • Creating interactivity like clicks, slides, and toggles
  • Form validation before submission
  • Showing/hiding content dynamically
  • Fetching data using APIs (AJAX)
  • Building games, calculators, and full web apps
  • Adding animations, alerts, and modals

Advantages of JavaScript:

  • Fast execution in the browser
  • Reduces server load by handling logic client-side
  • Cross-platform and supported by all browsers
  • Used on both frontend and backend (Node.js)
  • Massive ecosystem of libraries and frameworks

More Info:

Event-driven: JS responds to actions like clicks or typing.

Not Java: Despite the name, it's not related to Java language.

ES6+ brings modern features like arrow functions, modules, and classes.

Common JavaScript Tags and Methods:

  • <script>: Embeds JavaScript code into an HTML document
  • console.log(): Outputs a message to the console
  • document.getElementById(): Returns an element by its ID
  • document.getElementsByClassName(): Returns elements with a specified class name
  • document.querySelector(): Returns the first element that matches a specified CSS selector
  • document.querySelectorAll(): Returns all elements that match a specified CSS selector
  • window.alert(): Displays an alert box
  • window.confirm(): Displays a dialog box with OK and Cancel buttons
  • window.prompt(): Displays a prompt asking for user input
  • setTimeout(): Executes a function after a specified delay
  • setInterval(): Executes a function at specified intervals
  • clearTimeout(): Cancels a timeout
  • clearInterval(): Cancels an interval
  • Math.random(): Returns a random number between 0 and 1
  • Math.floor(): Returns the largest integer less than or equal to a number
  • Math.ceil(): Returns the smallest integer greater than or equal to a number
  • parseInt(): Converts a string into an integer
  • parseFloat(): Converts a string into a floating-point number
  • Array.push(): Adds one or more elements to the end of an array
  • Array.pop(): Removes the last element from an array
  • Array.shift(): Removes the first element from an array
  • Array.unshift(): Adds one or more elements to the beginning of an array
  • Array.map(): Creates a new array by applying a function to each element
  • Array.filter(): Creates a new array with elements that pass the test
  • JSON.parse(): Converts a JSON string into a JavaScript object
  • JSON.stringify(): Converts a JavaScript object into a JSON string
  • localStorage.setItem(): Saves data to local storage
  • localStorage.getItem(): Retrieves data from local storage
