CSS Tags (Properties) - Quick Guide

What is CSS?

CSS is the language used to style and design webpages. While HTML provides the structure, CSS controls how everything looks and feels—like colors, fonts, layouts, spacing, and animations. It's what turns a basic HTML page into a visually appealing and user-friendly website.

Uses of CSS:

  • Styling text: fonts, sizes, colors
  • Creating layouts: grids, columns, spacing
  • Adding background colors, images, and gradients
  • Making webpages responsive for phones and tablets
  • Creating hover effects, transitions, and animations
  • Hiding or showing elements based on screen size

Extra Information About CSS:

  • Inline: Applied directly to elements (not recommended for big projects)
  • Internal: Written within the HTML document
  • External: Linked as a separate file (most common and best practice)

Why "Cascading"?

The word "Cascading" means styles can override each other based on priority. For example, inline styles can override external styles unless managed using specificity and importance.

Role in Web Development:

  • Separates content (HTML) from design (CSS)
  • Makes code cleaner and easier to manage
  • Allows reusable styles across multiple pages
  • Enables responsive and mobile-friendly design

CSS Properties (with Examples):

  • color – color: red;
  • background-color – background-color: yellow;
  • font-size – font-size: 20px;
  • font-family – font-family: Arial;
  • font-weight – font-weight: bold;
  • text-align – text-align: center;
  • margin – margin: 10px;
  • padding – padding: 20px;
  • border – border: 1px solid black;
  • width – width: 300px;
  • height – height: 200px;
  • display – display: flex;
  • position – position: absolute;
  • top – top: 50px;
  • left – left: 100px;
  • right – right: 0;
  • bottom – bottom: 10px;
  • z-index – z-index: 10;
  • overflow – overflow: scroll;
  • box-shadow – box-shadow: 2px 2px 5px gray;
  • text-shadow – text-shadow: 1px 1px black;
  • border-radius – border-radius: 10px;
  • opacity – opacity: 0.5;
  • visibility – visibility: hidden;
  • background-image – background-image: url('img.jpg');
  • flex – flex: 1;
  • align-items – align-items: center;
  • justify-content – justify-content: space-between;
  • grid-template-columns – grid-template-columns: 1fr 2fr;
  • gap – gap: 10px;
  • cursor – cursor: pointer;
  • transition – transition: all 0.3s ease;
  • animation – animation: slide 2s infinite;
