HTML Basics: Important Tags

What is HTML?

HTML (HyperText Markup Language) is the foundational language used to create webpages. It structures and organizes the content on websites, such as text, images, links, videos, and more.

HTML works together with CSS (for styling) and JavaScript (for interactivity). However, HTML is essential because it gives meaning and order to the content.

Uses of HTML:

  • Creating webpages and web applications
  • Structuring content like headings, paragraphs, and images
  • Linking between pages and websites
  • Embedding multimedia such as videos and audio
  • Forming the base layout for modern websites
  • Helping search engines understand webpage content

Tags in HTML

<html> – Root element of an HTML document.
<head> – Contains metadata and links to styles/scripts.
<title> – Sets the page title shown in the browser tab.
<meta> – Provides metadata like character set or viewport settings.
<link> – Links external stylesheets or icons.
<style> – Adds internal CSS styles.
<script> – Embeds or links to JavaScript code.
<body> – Contains all the visible page content.
<header> – Defines page or section headers.
<nav> – Contains navigation links.
<main> – Main content of the document.
<section> – Groups related content into sections.
<article> – Represents an independent piece of content.
<aside> – Side content like ads or related links.
<footer> – Footer section for copyright, links, etc.
<h1> to <h6> – Headings, from largest to smallest.
<p> – Defines a paragraph.
<a> – Creates a hyperlink to another page or URL.
<img> – Embeds an image in the document.
<br> – Inserts a line break.
<hr> – Adds a horizontal line (thematic break).
<ul> – Unordered list with bullets.
<ol> – Ordered list with numbers.
<li> – List item inside a list.
<table> – Creates a table.
<tr> – Defines a table row.
<td> – Table data/cell.
<th> – Table header cell.
<form> – Creates an input form.
<input> – Accepts user input in a form.
<label> – Defines label for a form input.
<button> – Clickable button element.
<div> – Container for content, used for layout.
<span> – Inline container for styling text.
